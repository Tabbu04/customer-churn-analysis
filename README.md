# Customer Churn Analysis

Analyzed telecom customer churn using Python and Pandas to identify 
key factors driving customer attrition.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Telco Customer Churn Dataset (Kaggle) — 7032 customers, 21 features

## Key Insights
- Overall churn rate is ~26%
- Month-to-month contract customers churn the most
- Churned customers pay higher monthly charges on average
- New customers (low tenure) are more likely to churn
- Long term contracts (1-2 year) have the lowest churn rate
